# Lung Noom

“พวกเราได้ชักชวน ‘ลุงหนุ่ม’ เข้ามา เขาสามารถหาเสบียงที่อร่อยให้เราได้ และสามารถส่งเสบียงได้ภายใน 1 วินาที เขาคือบุคคลสำคัญในการหาเสบียงให้เรา”

# P.Petch

“พวกเราได้ชักชวน ‘พี่เพชร’ เข้ามา เขาเป็นฝ่ายหาข้อมูลต่างๆมาให้เราได้และสามารถหลอกล่อทีมอื่นๆให้เรา......
เขาคือบุคคลสำคัญในการหาข้อมูลและหลอกล่ออีกฝ่ายให้เรา”

# P.Leaf

“พวกเราได้ชักชวน ‘พี่ลีฟ’ เข้ามา เขาสามารถเล่นมุขเพื่อให้เราหายเครียดจากงาน และ หลอกล่ออีกฝ่ายได้เช่นกัน เขาคือบุคคลสำคัญในการรบกวนอีกฝ่ายได้อย่างสุดยอด”

# อจ.โอฬาร

“พวกเราได้ชักชวน ‘อจ.โอฬาร’ เข้ามา เขาสามารถทำให้ทุกคนหวาดกลัวได้เมื่อกล่าวถึงนามของเค้า เขาคือบุคคลสำคัญในการทำให้อีกฝ่ายหวาดกลัวกับแก๊งค์ของเรา”

# ห่านแห่งKmutt

“พวกเราได้ชักชวน ‘ห่านแห่งkmutt’ เข้ามา เขามีความสามารถในการต่อสู้ และ ทำให้อีกฝ่ายหวาดกลัวเมื่อคำราม เขาคือบุคคลสำคัญเมื่อมีเหตุการที่ต้องต่อสู้กับแก๊งค์อื่น”

# เหี้ยมแห่งKmutt

“พวกเราได้ชักชวน ‘เหี้ยมแห่งKmutt’ เข้ามา เขาสามารถลักลอบแอบเข้าตามสถานที่ต่างๆได้อย่างแนบเนียน 
เขาคือบุคคลสำคัญเมื่อมีเหตุการที่จะต้องลักลอบเข้าไปขโมยหรือหาข้อมูลต่างๆในแก๊งค์อื่น”

# น้อนแมว

“พวกเราได้ชักชวน ‘น้อนแมว’ เข้ามา เขาสามารถใช้ความน่ารักในการหลอกล่อคนในแก๊งค์อื่นได้
เขาคือบุคคลสำคัญเมื่อมีเหตุการที่จะต้องลักลอบเข้าไปขโมยหรือหาข้อมูลต่างๆในแก๊งค์อื่นเพื่อช่วยเหี้ยมแห่งKmuttให้ทำงานง่ายขึ้น”
